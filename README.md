# LODVault

LODVault is a lightweight browser extension for [lod.lu](https://lod.lu/).

## What it does
- Save words to Favorites or Study
- Add notes
- Search your saved words
- Review with flashcards
- Preview and export as HTML
- Import and export as JSON

## Install
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder

## Use
1. Open a word page on `lod.lu/artikel/*`
2. Save it from the page banner or popup
3. Open the popup to manage notes and lists
4. Use Preview, Flashcards, HTML, or JSON when needed

## Storage
All data is stored locally with `chrome.storage.local`.
